Welcome to the Immune Cells Classifier!

Immune Cells Classifier (ICC) is a free web application to classify cells from single cell RNA sequencing experiments into immune cells types.
Upload an expression data file (up to 200MB) and the classifier will classify the type of each cell.

ICC currently recognizes 11 immune cell types. More cell types will be added shortly.
Recognized cell types:
  • B
  • B1ab
  • DC
  • GN
  • MF
  • NK
  • NKT
  • T4
  • T8
  • Tgd
  • Treg
  • Other

Expression Data

The expression data file should contain genes (rows) and cells (columns). Cell Xij of the file will hold the expression level of gene i in cell j.
Note: The system currently supports expression files of up to 200MB.
Here is an example of such file:


When uploading an expression data file, note the following details:
1. The platform that was used to collect the data (Microarray/Bulk RNA Sequencing/Single Cell RNA Sequencing)
2. Expression data start row
3. Expression data end row (Default is the last line of the file)
4. Is the first row labeled? (cells labels)
5. Is the second row titled? (cells titles)
6. Are the genes labeled by gene symbol? If not, upload a conversion file as well

Conversion File

In case the genes in the expression data are not labeled by gene symbol, a conversion file should be uploaded.
Here is an example of a conversion file:


When uploading a conversion file, note:
1. Conversion data start row
2. Conversion data end row (Default is the last line of the file)
3. Gene ID column
4. Gene symbol column

Keep in mind that for full conversion of the genes, all genes of the expression data file must appear in the conversion file.

Conversion file tutorial:

Expression example file: GSE70941_bulk_expression
Expression file details:
Platform: Bulk RNA Sequencing
Data start row: 3
Data end row: 22298 (optional)
Genes labeled by gene symbol: No

Conversion example file: GSE70941_bulk_conversion
Conversion file details:
Conversion start row: 2
Conversion end row: 63815 (optional)
Gene ID column: 1 (default)
Gene symbol column: 2 (default)
First row labeled: No
Second row titled: No

Classification Methods Details

Classification results depends on the classification method. The default method is Random Forest.
For further explanation of the different methods:
  • Random Forset
  • KNN (K-nearest Neighbors)
  • SVM (Support-vector Machine)

Results Screen Interpretation

Classification results include, on the left, a pie-chart of the distribution of cell types. Upon hover the amount and percentages of that type is presented.
On the right, a bar chart of the distribution of confidence levels – upon hover the number of cells in the given confidence level is presented.
