The classifier will calculate a predicted type for each cell of the given input file. There are 11 classified cell types, as follows:
  • B
  • B1ab
  • DC
  • GN
  • MF
  • NK
  • NKT
  • T4
  • T8
  • Tgd
  • Treg
  • Other

Expression Data

The expression data file will contain genes (rows) and cells (columns). Each inner value of the file will hold the expression level of the given gene in the given cell.
Note: The system supports expression files of up to 200MB.
Here is an example of such file:


When uploading an expression data file, figure out the following details:
1. The platform that was used to collect the data (Microarray/Bulk RNA Sequencing/Single Cell RNA Sequencing)
2. Expression data start row
3. Expression data end row (If no value was given – will default choose the last line of the file)
4. Is the first row labeled? (cells labels)
5. Is the second row titled? (cells titles)
6. Are the genes labeled by gene symbol? If not, upload a conversion file as well

Expression file tutorial:

Example file: GSE116347_bulk_mouse_expression
File details:
Platform: Bulk RNA Sequencing
Data start row: 3
Data end row: 52999 (optional)
Genes labeled by gene symbol: Yes
First row labeled: Yes
Second row titled: Yes

Conversion File

In case the genes in the expression data are not labeled by gene symbol, a conversion file should be attached.
Here is an example of a conversion file:


When uploading a conversion file, figure out:
1. Conversion data start row
2. Conversion data end row (If no value was given – will default choose the last line of the file)
3. Gene ID column
4. Gene symbol column

Keep in mind that for full conversion of the genes, all genes of the expression data file must appear in the conversion file.

Conversion file tutorial:

Expression example file: GSE70941_bulk_expression
Expression file details:
Platform: Bulk RNA Sequencing
Data start row: 3
Data end row: 22298 (optional)
Genes labeled by gene symbol: No

Conversion example file: GSE70941_bulk_conversion
Conversion file details:
Conversion start row: 2
Conversion end row: 63815 (optional)
Gene ID column: 1 (default)
Gene symbol column: 2 (default)
First row labeled: No
Second row titled: No
